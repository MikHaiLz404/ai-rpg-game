# ระบบต่อสู้: Combat System

---

## ภาพรวม

ในโลก **Elzeria** การต่อสู้เป็นส่วนสำคัญของชีวิตประจำวัน ทั้งจากสัตว์ประหลาด มอนสเตอร์ หรือแม้แต่การปะทะกันระหว่างผู้เล่นด้วยกันเอง

ระบบต่อสู้ในโลกนี้ใช้หลัก **Attributes (ค่าสถานะ)** และ **Skills (ทักษะ)** ผสมผสานกัน

---

## ค่าสถานะ (Attributes)

| ค่าสถานะ | คำอธิบาย |
|----------|----------|
| **HP**   | จุดเลือด - เมื่อเหลือ 0 จะหมดสติหรือตาย |
| **MP**   | จุดเวท - ใช้สำหรับทักษะเวทมนตร์ |
| **STR**  | พลังแรง - ความแข็งแรงทางกายภาพ |
| **DEF**  | ป้องกัน - ลดความเสียหายจากการโจมตี |
| **AGI**  | ความว่องไว - ลำดับก่อน-หลังในการต่อสู้ |
| **INT**  | สติปัญญา - พลังเวทมนตร์และทักษะ |

---

## วิธีการต่อสู้

### 1. โจมตีปกติ (Basic Attack)
- ใช้อาวุธหรือกำปั้นโจมตีศัตรู
- ความเสียหาย = `STR × 1.0 + ค่าอาวุธ`
- ใช้เวลา 1 เทิร์น

### 2. ทักษะ (Skills)
- ต้องเรียนรู้จาก Skill Book หรือ Level Up
- แบ่งเป็น:
  - **Active Skills** - ต้องใช้ MP ในการใช้
  - **Passive Skills** - ทำงานอัตโนมัติ

### 3. เวทมนตร์ (Magic)
- ต้องมี Element ที่เข้ากันได้
- ใช้ MP ในการร่าย
- แบ่งเป็นธาตุต่างๆ: ไฟ, น้ำ, ลม, ดิน, ฟ้า, มืด, แสง

### 4. การป้องกัน (Defend)
- ลดความเสียหายลง 50%
- ใช้เวลา 1 เทิร์น

### 5. หนี (Run)
- มีโอกาสสำเร็จ 50% (ขึ้นกับ AGI)
- หากล้มเหลว เทิร์นจะจบโดยไม่ได้ทำอะไร

---

## สูตรความเสียหาย

### โจมตีปกติ
```
ความเสียหาย = (STR × 1.0 + ค่าอาวุธ) - (DEF × 0.5)
```

### เวทมนตร์
```
ความเสียหาย = (INT × 1.5 + พลังเวท) × สัมประสิทธิ์ธาตุ - (DEF ทางเวท × 0.5)
```

### คริติคอล
- โอกาส = `AGI × 0.5%`
- ความเสียหาย × 2

---

## ระบบเทิร์น (Turn System)

การต่อสู้ใช้ระบบเทิร์น:

1. **คำนวณลำดับ** - ตามค่า AGI
2. **ผู้เล่นเลือกการกระทำ** - โจมตี/ใช้ทักษะ/ป้องกัน/หนี
3. **ดำเนินการ** - คำนวณความเสียหาย
4. **ศัตรูตอบโต้** - ทำซ้ำข้อ 2-3

---

## การต่อสู้แบบเปิดเผย (Open Combat)

นอกจากการต่อสู้แบบเทิร์นแล้ว ใน **Gods' Arena** ยังมีรูปแบบการต่อสู้อื่นๆ:

### PvP (Player vs Player)
- สู้กันตรงๆ ไม่มีกฎเกณฑ์
- ใช้ได้ทุกวิธี

### Arena Match
- สนามรบมีกฎเกณฑ์
- ห้ามใช้ Item ระหว่างแข่ง
- ห้ามฆ่าคู่ต่อสู้

### Boss Raid
- รวมกลุ่มกันสู้บอส
- แบ่งปันของรางวัลตาม contribution
