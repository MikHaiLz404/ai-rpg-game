# บทที่ 3: เปิดร้านค้า

> ⏩ **จากบทที่ 2:** มินจูขายของครั้งแรกที่ตลาดกลางคืนสำเร็จ! เธอได้เงิน 3 เหรียญ และตั้งเป้าหมายจะเปิดร้านค้าของตัวเอง

---

หนึ่งเดือนผ่านไป

มินจูเปลี่ยนไปมาก—จากเด็กผู้หญิงข้างถนน กลายเป็นแม่ค้าเล็กๆ ที่มีลูกค้าประจำ

เธอใช้เงินที่หามาได้เช่าร้านเล็กๆ หลังคามุงจาก อยู่ตรงหัวมุมตลาด

"ร้านมิ้นท์" ป้ายชื่อเขียนด้วยลายมือติดอยู่ด้านบน

"วันนี้จะขายอะไรดีน้อ?" มินจูพึมพำ จัดของบนชั้น

เธอมีของหลายอย่าง—หม้อ กระจก ผ้า เชือก มีด และของแปลกๆ ที่เธอคิดค้นขึ้นเอง

"ลูกค้าคนแรกวันนี้จะเป็นใครนะ?" เธอคิด

*กระดิ่งดัง*

"ยินดีต้อนรับค่ะ!"

มินจูหันไปมอง หัวใจเต้นแรง

ชายร่างสูงใหญ่ยืนอยู่ที่ประตู สวมเกราะสีเงินเป็นมันวาว ผมทองยาวพลิ้วไหว ตาสีแดงเข้มจ้องมองเธอ

"หืม..." เขาพูดเสียงทุ้ม "ร้านเล็กจัง"

"ค่ะ!" มินจูตอบทันที "ร้านเล็กแต่ของดีค่ะ! ท่านลูกค้าต้องการอะไรคะ?"

ชายมองไปรอบๆ ร้าน

"กระจก" เขาพูด "ต้องการกระจก"

"กระจกค่ะ!" มินจูรีบไปหยิบกระจกเล็กๆ มาให้ "ใบนี้สามเหรียญค่ะ แวววาวมากเลย!"

ชายรับกระจกมา ส่องดูใบหน้าตัวเอง

"อืม..." เขาพึมพำ "ก็ได้"

เขาวางเงินสามเหรียญลงบนเคาน์เตอร์ แล้วจะเดินออกไป

"รอก่อนค่ะ!" มินจูตะโกน

"อะไร?" ชายหันกลับมา

"ท่าน... ท่านไม่ใช่คนธรรมดาใช่ไหมคะ?" เธอถาม

ชายหยุดนิ่ง

"ทำไมถาม?"

"เพราะ..." มินจูกลืนน้ำลาย "กระจกใบนี้ขายสามเหรียญ แต่ท่านให้มาเลยห้า หนูต้องทอนให้ค่ะ"

"... ไม่ต้องทอน"

"แต่—"

"ไม่ต้อง" ชายพูดชัด "ถือไป"

แล้วเขาก็เดินออกไป

มินจูยืนงงอยู่ครู่หนึ่ง

```

│  ✅ ลูกค้าพิเศษ!                    │

│  - ซื้อ: กระจก (3 เหรียญ)           │

"เทพ..." มินจูกระซิบ "ท่านนั่นเป็นเทพ!"

เธอรีบปิดประตู หันไปมองระบบ

"เอาล่ะ แต้มสะสมสามสิบแล้ว! ซื้ออะไรดีน้อ?"

เธอเปิดเมนูแลกเปลี่ยน

```
```
✅

```

"อืม..." มินจูคิด "ซื้อลูกแก้วนำโชคดีกว่า ราคาสูงแต่คุ้ม!"

เธอแลกลูกแก้วมา

ลูกแก้วใส ขนาดเท่าไข่นกกระทา เปล่งประกายเล็กๆ

"สวยจัง..." เธอกระซิบ

แต่แล้วเธอก็นึกขึ้นได้

'รอก่อน ถ้าท่านเทพนั่นกลับมาอีกล่ะ? ฉันต้องเตรียมของดีๆ ไว้!'

เธอรีบจัดของในร้านใหม่ หยิบกระจกใบใหญ่ขึ้นมาวางไว้ตรงกลาง เพิ่มผ้าเช็ดหน้าสีสวยๆ

"เสร็จแล้ว!" เธอพยักหน้า "พรุ่งนี้เปิดร้านใหม่!"
